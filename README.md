# Mern Ecommerce Public
__This is my first mern e-commerce!__

## Quickstart

1.Install dependencies in client:
```bash
  cd client
  npm install
```
2.Install dependencies in server:
```bash
  cd server
  npm install
```
3.Set it up settings in .env
```
PORT = 8080
JWT_SECRET = YOUR_JWT_SECRET
MONGO_URL = 'YOUR_MONGO_URL'
```
